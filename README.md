# Watchtower

Watchtower is a modern Java Plugin for PaperMC Servers that adds a visual clock to the game.

# Usage

Toggle the clock with 
````
/watch
````

# Compatibility

The Watchtower Plugin is compatible with Servers built for 1.20 and above
